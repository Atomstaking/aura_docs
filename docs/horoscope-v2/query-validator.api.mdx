## Query validator

Query:
```graphql
query queryValidator(
  $offset: Int = 0
  $limit: Int = 10
  $operatorAddress: String = null
) {
  xstaxy {
    validator(
      limit: $limit
      offset: $offset
      order_by: { tokens: desc }
      where: { operator_address: { _eq: $operatorAddress } }
    ) {
      account_address
      commission
      consensus_address
      consensus_hex_address
      created_at
      consensus_pubkey
      delegator_shares

      }

        count
      }
    }
  }
}
```
